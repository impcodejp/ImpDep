# ImpDep

## 1. 概要
ImpDepは、案件の進捗や予算状況の可視化、顧客設備の詳細な管理をワンストップで行うためのデスクトップアプリケーションです。
日々の業務におけるプロジェクト管理や、ハードウェア・ソフトウェアの資産把握をスムーズにサポートします。

## 2. 動作環境
- **対応OS**: Windows 10 / Windows 11 (64bit)
- **必須コンポーネント**: Microsoft Edge WebView2 
  *(※通常、Windowsに標準でインストールされています。)*

## 3. インストール手順
1. 配布されたインストーラー（例：`ImpDep_Setup.exe`）をダウンロードします。
2. インストーラーをダブルクリックして起動します。
   *(※「WindowsによってPCが保護されました」という青い画面が出た場合は、「詳細情報」をクリックしてから「実行」を選択してください)*
3. 画面の指示に従ってインストールを完了させます。
4. デスクトップに作成された「ImpDep」のアイコンからアプリを起動してください。

## 4. 主な機能と使い方

### 📊 ダッシュボード・全体把握
- **全体の可視化**: 現在のアクティブな案件数や、売上・粗利の状況を一目で確認できます。
- **実稼働の集計**: 契約稼働値が「0」の案件は自動的に集計から除外されるため、実際に動いているプロジェクトだけを正確に把握できます。
- **スケジュールの確認**: 画面右側にはプロジェクトのスケジュール表が配置されており、見やすいように幅のサイズ調整が可能です。
- **画面の有効活用**: 左側のメニューコンポーネントは折りたたみ可能（初期状態は閉じた状態）なため、画面を広く使って作業ができます。

### 📁 案件計上管理
- **案件の登録・編集**: プロジェクトの基本情報や目標を登録できます。
- **対応履歴（変更ログ）**: プロジェクト内で発生した変更履歴や対応のログを残すことができます。

### 🏢 顧客・設備管理
- **顧客情報の登録**: クライアントの基本情報や連絡先をリスト化して管理します。
- **利用資産の管理**: 顧客ごとに導入されている「ハードウェア情報」「ソフトウェア情報」、および「他システム情報」を詳細にひもづけて記録できます。

### 💰 予算・レポート機能
- **予算設定**: 目標となる予算を設定し、残額や達成状況をトラッキングします。
- **負荷推移レポート**: 担当者や期間ごとの稼働の推移をレポートとしてグラフなどで出力・確認できます。

### ✅ ToDo管理
- **ToDoリストの作成**：業務内で発生したToDoの登録／消込が可能です。

## 5. よくあるご質問（トラブルシューティング）

**Q. アプリが起動せず、真っ白な画面になる**
A. ご利用のPCに「Microsoft Edge WebView2」がインストールされていない、または古いバージョンの可能性があります。Windows Updateをご確認いただくか、サポート担当までご連絡ください。

**Q. 画面の表示が崩れてしまう**
A. アプリのウィンドウサイズを広げるか、左側のメニューを折りたたんで表示領域を確保してみてください。売上や粗利の数字なども、桁数が多くても崩れないよう調整されています。
